Pyecharts_Candlestick

2018-12-12 17:43:06

pyecharts¶

pyecharts是echarts的python套件，而echarts則是百度公開的javascript工具，用來做資料視覺化。以下以葡萄王（1707）來做示範。¶

In [2]:
from pandas_datareader import data
from pyecharts import Kline, Line, Overlap, Bar, Grid, Scatter
from datetime import datetime
import numpy as np
from pyecharts import online, Page

online()

加上online()，jupyter notebook輸出成html，才會能看到圖¶

Get Data¶

之前padas可以透過pandas.io.data的web.DataReader直接拿到股票資料，不過現在改成要透過pandas_datareader的data.DataReader來取得。Kline吃的資料格式是list，預設的資料順序是'Open', 'Close', "High", 'Low'。¶

In [3]:
df = data.DataReader('1707.TW', 'yahoo', datetime(2016, 10, 1), datetime.now())
v1 = df[['Open', 'Close', "High", 'Low']].values

K線圖¶

In [4]:
kline = Kline()
kline.add("日K", df.index, v1)
Out[4]:

加上成交量¶

In [5]:
bar = Bar()
bar.add("",
    df.index,
    df.Volume,
    is_xaxislabel_align=True,
    legend_top="70%",
    legend_pos="left",
    xaxis_pos="bottom",
)

grid = Grid(width=800, height=600)
grid.add(bar, grid_top="70%")
grid.add(kline, grid_bottom="35%")

加個移動平均線¶

In [6]:
line = Line()
line.add("20MA", df.index.astype('str'), df.Close.rolling(window=20).mean())
