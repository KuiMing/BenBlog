In [1]:
from datetime import datetime
import sys
import os

datetime和sys不用多說了，一個是為了時間，一個是為了讓我可以直接從terminal拿到參數。第三個os則是為了讓我可以直接把osx command用python發出去，會需要用到os.system，十分直觀。¶

剩下的幾乎就跟用Ｒ一樣了，產生一個以當下時間命名的資料夾，然後把要發文的內容全部丟到該資料夾，就結束了。¶

In [2]:
def mv_for_blog(files):
    today=datetime.today()
    repo=today.strftime("%Y-%m-%d_%H-%M-%S")
    command = 'mkdir '+repo
    os.system(command)
    command = 'mv '+files+'* '+repo
    os.system(command)

if len(sys.argv) < 2:
    print 'no argument'
    sys.exit()
else:
    mv_for_blog(sys.argv[1])
