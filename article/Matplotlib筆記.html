Matplotlib筆記

2016-12-26 12:40:13

matplotlib in ipython¶

畫出圖形存圖¶

In [3]:
from pylab import figure, plot, savefig
import numpy as np
figure(figsize=(15,10))
x = np.arange(0,180)
y = np.sin(x * np.pi / 180.0)
plot(x,y)
savefig('matplotlib.png')

直接畫在ipython notebook上¶

In [7]:
import matplotlib
%matplotlib inline
matplotlib.style.use('ggplot')

使用magic function%呼叫matplotlib 之中的inline後，發現matplotlib多出了style，可以讓圖形有ggplot2的風格。¶

In [12]:
figure(figsize=(15,10))
x = np.arange(0,180)
y = np.sin(x * np.pi / 180.0)
plot(x,y)
savefig('matplotlib.png') #依然可以存圖

不過上緣被切掉一點，調整ylim。¶

In [15]:
from matplotlib.pylab import ylim
In [16]:
figure(figsize=(15,10))
x = np.arange(0,180)
y = np.sin(x * np.pi / 180.0)
z = np.cos(x * np.pi / 180.0)
ylim(-1.1,1.1)
plot(x,y)
plot(x,z) #順手多畫一條cos的線。
